# PortfolioHUB

## Sobre o Repositório

Olá, meu nome é Pedro Henrique e este repositório foi desenvolvido como parte da atividade acadêmica “Criação de Repositório com Versionamento”.

O objetivo deste projeto é organizar trabalhos acadêmicos e projetos pessoais de forma estruturada utilizando Git e GitHub, aplicando conceitos de versionamento, documentação e compartilhamento online.

Além disso, o projeto também busca fortalecer o perfil profissional através da integração com plataformas como GitHub e LinkedIn.

---

## Objetivo da Atividade

Este repositório foi criado com o objetivo de:

praticar versionamento com Git;
organizar projetos acadêmicos;
desenvolver documentação utilizando README.md;
publicar projetos online utilizando GitHub Pages;
integrar o projeto ao LinkedIn.
